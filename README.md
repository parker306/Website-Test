# Website-Test

A prototype for hosting the specimen using ScaleFlex cloudimage-360-view. 

## How to use:
Copy the files to the location you want to host it.

## Example layout
Theres a nested folder and the xmls files that are being used as an example on how the layout should be placed to allow the javascripts to read them.

## How to add more specimen categories
add more specimen categories to the "Types.xml" by opening it via excel and enter more into the rows.

Add new folder inside the gallery folder for the new type with the name of the category code. Inside the new folder there should: 
- xml with the same category code.
- folder called "thumb".
	
## How to add new specimens
Make a new folder inside the correct category with the name of the specimen code, inside the folder it needs:
- images of the specimen with the naming convention "CodeName - (number)" e.g. "A1 - (1)".
- xml file with the code name as the filename.
	
Inside the "thumb" folder it needs a thumbnail image of the specimen.
